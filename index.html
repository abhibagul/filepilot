FilePilot Client

A high-performance dual-pane desktop application built with Rust and Tauri. Native performance, low memory footprint, and robust local credential sealing.

[ ZOOM ]
  • Dual-pane directory structure with synchronized browsing
  • Zero-Knowledge local credential sealing (PBKDF2 / AES-256)
  • Folder syncing with live side-by-side diff viewers
  • Offline transfer queuing and auto-resume engine

FilePilot Enterprise Vault

A centralized Express/Node.js provisioning server that secures and audits connection profiles across platform and database administration teams.

[ ZOOM ]
  • Centralized connection profile token-scoped provisioning
  • Blockchain-style tamper-evident audit logs (SHA-256 chain)
  • Multiple Database backends (SQLite, PostgreSQL, MySQL)
  • Real-time token revocation and remote file restoration

Multi-Protocol Integration

FilePilot supports all industry-standard transfer protocols out-of-the-box, optimized via Rust's async runtime engine.

SFTP (SSH)

Secure Shell File Transfer with password & SSH key pair authentication, SOCKS proxies, and jump-host chaining.

Amazon S3

Native AWS bucket browser, bucket actions, custom endpoints (MinIO/Ceph), and concurrent multipart transfers.

FTPS (TLS/SSL)

Classic FTP wrapped in modern transport-layer security with implicit/explicit modes and validation certificates.

WebDAV

HTTP-based authoring and versioning, optimized for private network attached storage (NAS) and Cloud mounts.

Need a proprietary protocol? Build it using the FilePilot Plugin SDK →
