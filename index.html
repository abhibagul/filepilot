Connects to Just About Anything

SFTP, FTP, S3, WebDAV, cloud drives and more — connect to all of them from one window.

SFTP (SSH)

Secure Shell transfers with password or SSH key-pair auth, SOCKS proxies, and jump-host chaining.

FTP

Standard FTP with active and passive modes — works through firewalls and NAT without extra setup.

FTPS (TLS/SSL)

Classic FTP wrapped in full TLS with implicit/explicit modes and certificate validation.

SCP

Blazing-fast file copy over SSH tunnels for quickly pushing files without a full session.

Amazon S3

Browse buckets like a filesystem — regions, custom endpoints (MinIO/Ceph), multipart transfers.

WebDAV

Nextcloud, ownCloud, NAS devices, and any HTTP-based file server with full read/write access.

Cloud Storage

First-class Google Drive, Dropbox, OneDrive, and Azure Blob support in the same dual pane.

Proxy / Jump Host

Tunnel through SOCKS5 or HTTP proxies, or chain SSH bastions to reach private networks.

How Team Access Works

How your team gets access to servers — without a password ever changing hands.

1

Provision

An admin defines connection profiles in the Vault. Secrets are wrapped with your cloud KMS (AWS, Azure, GCP, or HashiCorp) and never stored in plaintext.

2

Sync

The client redeems a scoped access token over a TLS WebSocket. Profiles are decrypted in-memory and re-sealed into the local encrypted database.

3

Transfer & Audit

Every transfer runs directly against the server. Each action streams back to a tamper-evident SHA-256 audit chain the admin can verify.
